What is map()?

  • The map() method creates a fresh array by iterating over the elements of an existing array and applying a provided callback function to each element.
  • It doesnâ€™t modify the original array; instead, it constructs a new one based on the results of the callback function.
  • const newArray = originalArray.map(callbackFunction);
  1. originalArray: The array you want to transform.
  2. callbackFunction: A function that defines how each element should be transformed. It takes three arguments:
  3. currentValue: The value of the current element being processed.
  4. index (optional): The index of the current element.
  5. array (optional): The original array.
  • Beware of parseInt() with map():
